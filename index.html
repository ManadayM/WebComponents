Web Components Life Cycle

  • constructor() : Element is Created but not attached to DOM yet. Basic initializations only.
  • connectedCallback() : Called when your element has been attached to the DOM. You can do DOM initializations here.
  • disconnectedCallback() : Called when your element is detached from the DOM. Clean up work for e.g. cancel any ongoing HTTP requests.
  • attributeChangedCallback(): Called when observed attributes changed. Will be covered later.
